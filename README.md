הערות:

בהוספת חבר, מוסיפים את פרטיו האישיים בלבד בלי קשר לפרטי החיסונים שלו. רק לאחר שהחבר נוסף בהצלחה, ניתן להוסיף/לערוך/למחוק את חיסוניו. 

בוצעו בדיקות תקינות לקלטים הנדרשים.
